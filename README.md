# Flask Tasks API

Backend REST API built with Flask, MySQL and JWT authentication.

This project was created as practice for backend development and includes user authentication, database interaction and protected routes.

---

## Features

- User registration
- User login
- JWT token authentication
- Protected routes using Authorization header
- Create tasks for authorized users
- Get tasks only for current user
- MySQL database integration

---

## Technologies Used

- Python
- Flask
- MySQL
- mysql.connector
- PyJWT
